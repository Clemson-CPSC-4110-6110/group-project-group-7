This package contains water textures and foam masks. All textures are seamless. It has been tested in Unity 6 and 2022 versions, but since this package only includes textures, it should work in all Unity versions and all render pipelines (Built-in/URP/HDRP). It will work with any water system you use. All you need to do is replace the water texture or foam mask to achieve different looks. All textures are 2k resolution.
